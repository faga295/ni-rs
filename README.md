# ni-rs
`ni-rs` is rust implement of [ni](https://github.com/antfu/ni).

install it globally.
```
npm i -g ni-rs
```


`ni-rs` refers to [detect-package-manager](https://github.com/egoist/detect-package-manager)'s detect strategy.
## install
```
nirs

# npm install
# yarn install
# pnpm install
```
